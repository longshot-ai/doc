---
title: "FactGPT"
api: "POST https://api-v2.longshot.ai/custom/api/generate/factgpt/custom"
description: "This endpoint allows you to generate the factual data - which we call as FactGPT."
---

### Headers

<ParamField header="AUTHORIZATION" type="string" placeholder="Bearer <Secret Token>" required>
  Add the Authorization Bearer Token to authorize your api request.
</ParamField>

### Body

<ParamField body="instruction" type="string" required>
  This field is used as input the text as instructional prompt.
  
  Example: 
  "Generate a list of 5 frequently asked questions for this text:"Mumbai is a city with many bright tourist spots"" 
  OR
  "Bard: Google answer to ChatGPT"
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. success if successful, otherwise if not.
</ResponseField>

<ResponseField name="copies" type="object">

The contents of the copies

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the generated factual content.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="citations" type="string">
  This array provides the source / links of the sources which have been considered while writing the generated content.
</ResponseField>

<ResponseField name="message" type="string">
  This is the information attached to the call. This is useful when any error occurs, it provides more meaningful information about it. 
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits used for this particular request. It varies upon the generated content.
</ResponseField>
