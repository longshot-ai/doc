---
title: "Paragraph from Bullet Points"
api: "POST https://api-v2.longshot.ai/custom/api/generate/paragraph/bullet/points"
description: "This endpoint allows you to generate a paragraph from the bullet points."
---

### Headers

<ParamField header="AUTHORIZATION" type="string" placeholder="Bearer <Secret Token>" required>
  Add the Authorization Bearer Token to authorize your api request.
</ParamField>

### Body

<ParamField body="bullet_points" type="string" required>
  This field is used as input to provide an array of bullet_points.
  
  Example: 
  ["What is FOMO","What are the main causes of FOMO","How can you avoid FOMO"]
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. success if successful, otherwise if not.
</ResponseField>

<ResponseField name="copies" type="object">

The contents of the copies

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the paragraph content generated from the bullet points.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="message" type="string">
  This is the information attached to the call. This is useful when any error occurs, it provides more meaningful information about it. 
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits used for this particular request. It varies upon the generated content.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/paragraph/bullet/points' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
-H 'Content-Type: application/json' \
--data-raw '{
    "bullet_points": ""
}'
```

</RequestExample>

<ResponseExample>

```json Response
	"copies": [

		}

	"credits_used": <Credits>

```

</ResponseExample>
