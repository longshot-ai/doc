---
title: "Paragraph from Bullet Points"
api: "POST https://api-v2.longshot.ai/custom/api/generate/paragraph/bullet/points"
description: "This endpoint allows you to generate a paragraph from the bullet points."
---

### Headers

<ParamField header="Authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Input the Authorization Bearer Token to authorize your API request.
</ParamField>

### Body

<ParamField body="bullet_points" type="string" required>
  This field is used as an input to provide an array of bullet points to generate a paragraph.
  
  Example: 
  ["What is FOMO","What are the main causes of FOMO","How can you avoid FOMO"]
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. It prompts success if working, otherwise not.
</ResponseField>

<ResponseField name="copies" type="object">

The generated content.

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the paragraph generated from the bullet points.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="message" type="string">
  This information is attached to the call and offers meaningful details when errors occur.
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits consumed for this particular request. It varies on the generated content.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/paragraph/bullet/points' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
-H 'Content-Type: application/json' \
--data-raw '{
    "bullet_points": ""
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
	"copies": [
		{
			"content": "<Generated Paragraph>"
		}
	],
	"status": "success",
	"message": "Generated successfully",
	"credits_used": <Credits>
}
```

</ResponseExample>
