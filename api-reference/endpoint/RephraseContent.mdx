---
title: "Rephrase your content"
api: "POST https://api-v2.longshot.ai/custom/api/content/rephrase"
description: "This endpoint lets you rephrase your content passed as input."
---

### Headers

<ParamField header="authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Add the Authorization Bearer Token to authorize your api request.
</ParamField>

### Body

<ParamField body="text" type="string">
   This field is to input text you want to rephrase.
   
   Example:
  "Tesla has launched its coolest car yet. Equipped with the latest in tech, it is soon becoming a symbol of style among millennials and gen x alike."
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. success if successful, otherwise if not.
</ResponseField>

<ResponseField name="copies" type="object">

The contents of the copies

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the rephrased content generated from the api.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="text" type="string">
  The input text provided to be rephrased.
</ResponseField>

<ResponseField name="message" type="string">
  This is the information attached to the call. This is useful when any error occurs, it provides more meaningful information about it. 
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/content/rephrase' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
	"copies": [
		{
			"content": "<Rephrased Content>"
		}
	],
	"text": "<Input Text>",
	"status": "success",
	"message": "copies generated successfully"
}
```

</ResponseExample>
