---
title: "Rephrase your content"
api: "POST https://api-v2.longshot.ai/custom/api/content/rephrase"
description: "This endpoint lets you rephrase your content based on your input to convey your message better."
---

### Headers

<ParamField header="Authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Input the Authorization Bearer Token to authorize your API request.
</ParamField>

### Body

<ParamField body="text" type="string" required>
   This field is to input the text you want to rephrase.
   
   Example: 
   "Tesla has launched its coolest car yet. Equipped with the latest tech, it's becoming a symbol of style among Millennials and Gen X alike. And it's quite easy on the environment."
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. It prompts success if working, otherwise not.
</ResponseField>

<ResponseField name="copies" type="object">

The generated content.

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the rephrased content generated from the API.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="text" type="string">
  This is the input text you passed, to rephrase.
</ResponseField>

<ResponseField name="message" type="string">
  This information is attached to the call and offers meaningful details when errors occur. 
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/content/rephrase' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
	"copies": [
		{
			"content": "<Rephrased Content>"
		}
	],
	"text": "<Input Text>",
	"status": "success",
	"message": "copies generated successfully"
}
```

</ResponseExample>
