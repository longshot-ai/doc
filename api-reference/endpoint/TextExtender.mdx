---
title: "Text Extender"
api: "POST https://api-v2.longshot.ai/custom/api/generate/extend/content"
description: "This endpoint allows you to extend (or expand) your existing text or paragraph for better depth."
---

### Headers

<ParamField header="Authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Input the Authorization Bearer Token to authorize your API request.
</ParamField>

### Body

<ParamField body="text" type="string" required>
  This field inputs content you want to extend.
  
  Example: 
  "Mumbai is in the tropics, which means that it is very close to the equator and receives a lot of rainfall throughout the year. 
  The city's drainage network is inadequate, exacerbating the flooding from tropical cyclones."
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. It prompts success if working, otherwise not.
</ResponseField>

<ResponseField name="copies" type="object">

The generated content.

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the extended content generated from the API.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="text" type="string">
  This is the input text provided to the text extender.
</ResponseField>

<ResponseField name="message" type="string">
  This information is attached to the call and offers meaningful details when errors occur.
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits consumed for this particular request. It varies on the generated content.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/extend/content' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>
