---
title: "Text Extender"
api: "POST https://api-v2.longshot.ai/custom/api/generate/extend/content"
description: "This endpoint allows you to extend the text for more information about the content passed."
---

### Headers

<ParamField header="AUTHORIZATION" type="string" placeholder="Bearer <Secret Token>" required>
  Add the Authorization Bearer Token to authorize your api request.
</ParamField>

### Body

<ParamField body="TEXT" type="string">
  This field is used to input content you want to be extended.
  
  Example: 
  "Mumbai is in the tropics, which means that it is very close to the equator and receives a lot of rainfall throughout the year. 
  The city's drainage network is inadequate, exacerbating the flooding from tropical cyclones."
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. success if successful, otherwise if not.
</ResponseField>

<ResponseField name="copies" type="object">

The contents of the copies

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the extended content generated from the api.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="text" type="string">
  The input text provided to text extender.
</ResponseField>

<ResponseField name="message" type="string">
  This is the information attached to the call. This is useful when any error occurs, it provides more meaningful information about it. 
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits used for this particular request. It varies upon the generated content.
</ResponseField>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/extend/content' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>
