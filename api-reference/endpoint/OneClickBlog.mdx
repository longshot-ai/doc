---
title: "One Click Blog"
api: "POST https://api-v2.longshot.ai/custom/api/generate/one/click/blog"
description: "This endpoint enables you to craft a compelling, factually accurate blog in just one click."
---

### Headers

<ParamField header="Authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Input the Authorization Bearer Token to authorize your API request.
</ParamField>

### Body

<ParamField body="topic" type="string" required>
  This input field requires the user to mention the topic they wish to write on.
  
  Example: 
  "Lithium Ion discovery in India" 
  OR
  "Bard: Google answer to ChatGPT"
</ParamField>

<ParamField body="description" type="string" required>
  This input field requires the user to elaborate on the topic they wish to write on.
  
  Example: 
  "Showcase the future of manufacturing in India, following Lithium Ion discovery."
</ParamField>

<ParamField body="blog_type" type="string" default="listicle" required>
  This input field defines the type of blog you wish to generate.
  
  Following are the blog_types you can use
  1. how_to,
  2. listicle,
  3. pillar,
  4. comparison,
  5. product_review,
  6. product_alternative
</ParamField>

<ParamField body="mode" type="string" default="serp" required>
  This field is used to decide the mode of the generation. 

  Modes are as follows:
  1. serp
  2. domain
  
</ParamField>

<ParamField body="domain_list" type="string">
  This field is used to input the array of the domains you want to provide as a reference, while generating the content.
  This field needs to be passed if mode is "domain".
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. It prompts success if working, otherwise not.
</ResponseField>

<ResponseField name="copies" type="object">

The generated content.

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  Here's the one-click blog (HTML-formatted).
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="message" type="string">
  This information is attached to the call and offers meaningful details when errors occur.
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits consumed for this particular request. It varies on the generated content for different use cases.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/one/click/blog' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
-H 'Content-Type: application/json' \
--data-raw '{
    "topic": "",
    "description": "",
    "blog_type": "listicle",
    "mode": "serp",
    "domain_list": []
}'
```

</RequestExample>
