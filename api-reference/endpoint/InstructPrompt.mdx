---
title: "Instruct Prompt"
api: "POST https://api-v2.longshot.ai/custom/api/generate/instruct"
description: "This endpoint allows you to instruct the prompt for specific information about the content passed as input. Add escape sequences in text wherever necessary."
---

### Headers

<ParamField header="authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Add the Authorization Bearer Token to authorize your api request.
</ParamField>

### Body

<ParamField body="text" type="string">
  This field is used to Prompt instruction with the text.
  
  Example: 
  "5 popular food places for this text:\”Authentic Indian wine with the French style\”"
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. success if successful, otherwise if not.
</ResponseField>

<ResponseField name="copies" type="object">

The contents of the copies

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the rephrased content generated from the api.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="text" type="string">
  The input text provided to be rephrased.
</ResponseField>

<ResponseField name="message" type="string">
  This is the information attached to the call. This is useful when any error occurs, it provides more meaningful information about it. 
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/content/rephrase' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>

```json Response
	"copies": [

```

</ResponseExample>
