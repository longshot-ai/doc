---
title: "Instruct Prompt"
api: "POST https://api-v2.longshot.ai/custom/api/generate/instruct"
description: "This endpoint allows you to instruct or command AI to generate content based on your prompt. Add escape sequences in the text wherever necessary."
---

### Headers

<ParamField header="Authorization" type="string" placeholder="Bearer <Secret Token>" required>
  Input the Authorization Bearer Token to authorize your API request.
</ParamField>

### Body

<ParamField body="text" type="string" required>
  This field is used to input the prompt (text instruction) to generate results.
  
  Example: 
  “Mention 5 popular food places in Mumbai for this text:\” Authentic Indian wine with the French style\””
</ParamField>

### Response

<ResponseField name="status" type="string">
  Indicates whether the call was successful. It prompts success if working, otherwise not.
</ResponseField>

<ResponseField name="copies" type="object">

The generated content.

<Expandable title="Toggle object">

<ResponseField name="content" type="string">
  This is the response to the content generated from the instruction prompt.
</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="citations" type="string">
  This array provides the links of the sources considered while generating content.
</ResponseField>

<ResponseField name="message" type="string">
  This information is attached to the call and offers meaningful details when errors occur.
</ResponseField>

<ResponseField name="credits_used" type="number">
  This field informs you about the credits consumed for this particular request. It varies on the generated content.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api-v2.longshot.ai/custom/api/generate/instruct' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "text": ""
}'
```

</RequestExample>
